# 知乎文章：马可波罗到底有没有来过中国？我把他的路线做成交互地图后，有了答案

---

关于马可·波罗，中文互联网最常讨论的一个问题是：**他到底有没有来过中国？**

质疑派理由很充分：《马可·波罗行纪》里没提长城、没提筷子、没提喝茶、没提汉字。一个在中国住了17年的人，怎么可能不提这些？

我翻了一遍文献，又把他走过的路线做成了一套交互地图。做完之后我的判断是——**质疑派可能搞错了方向。马可·波罗书里"没提"的东西，恰恰更支持他来过。**

## 一本改变世界地图的书

先说说这本书有多厉害。

1298年，威尼斯和热那亚打仗。一个威尼斯商人在海战中被俘，关进了热那亚监狱。在狱中，他向狱友——一个叫鲁斯蒂谦的比萨小说家——口述了自己24年的东方游历。鲁斯蒂谦以法意混合语（Franco-Italian）记录下来，1298–1299年间成书，就是《马可·波罗行纪》（又译《东方见闻录》）。

这本书在当时引发的震动，大约相当于今天有人从外星球回来写了一本游记。

书里写了什么？

- 中国（契丹）的城市比欧洲任何一个城市都大、都富有
- 有一种黑色的石头（煤）可以像木柴一样燃烧，而且更持久
- 有一种叫"纸币"的东西——纸张印上印章就能当金银使
- 有一个叫"行在"（杭州）的地方，城里有12,000座桥
- 蒙古帝国的驿站一日四百里，从北京到威尼斯全程可以换马不换人

这些内容在14世纪的欧洲读者看来，基本等同于天方夜谭。

**但有一个人信了，而且信得很认真。**

两个世纪后，一个热那亚水手搞到了一本《马可·波罗行纪》。他在页边写满了批注，计算着向西航行到契丹和西潘戈（日本）的距离。这个水手叫克里斯托弗·哥伦布。他那本批注密布的《行纪》至今保存在塞维利亚的哥伦比亚图书馆。

1492年，哥伦布带着这本书出海。他向西航行，想要到达书中遍地黄金的东方——结果一头撞上了美洲大陆。

**一本书，改变了世界地图。** 这不是比喻，是事实。

## 他到底走了多远

我把马可·波罗的路线还原到了真实地图上。这是他的行程：

**东行（1271–1275）：**
威尼斯 → 阿卡 → 大不里士 → 起儿漫 → 霍尔木兹 → 翻越帕米尔高原 → 喀什 → 于阗 → 敦煌 → 河西走廊 → 上都（忽必烈夏宫）→ 大都（北京）

**在中国（1275–1292）：**
北京 → 大运河沿线 → 济南 → 扬州 → 苏州 → 杭州 → 泉州 → 云南 → 西藏（存疑）

**归航（1292–1295）：**
泉州 → 占城（越南）→ 苏门答腊 → 斯里兰卡 → 卡利卡特（印度）→ 霍尔木兹 → 君士坦丁堡 → 威尼斯

全程估算约 **22,000公里**。在当时，这是人类最长的连续旅行记录之一。

把这条路线画在地图上，再回头看那些质疑——很多问题就有了解释。

## 「他没提长城」—— 恰恰证明他来过

质疑派最大的论据是：马可·波罗没提长城。

**但元朝的长城根本是一段不需要提的残墙。**

我们今天看到的砖石长城，绝大多数是明代修建的。元代之前的秦汉土墙长城大部分已经坍塌，只剩低矮的土埂。元朝本身是蒙古人建立的，而蒙古人最不想要的就是一道隔开草原和中原的墙——长城在元代被刻意荒废，毫无军事意义。

一个在元朝生活了17年的人，路过一段不起眼的土埂，不觉得值得专门写下来——这不合理吗？反过来想一想：如果马可·波罗是坐在热那亚监狱里凭空编故事，他反而**一定会**把长城写进去。长城在欧洲传说中早已是传奇——一个靠想象写作的人，怎么可能漏掉读者最想看的奇观？

**真正到过中国的人，才会觉得长城不值一提。**

## 「他没提筷子和茶」—— 蒙古宫廷本就不用

马可·波罗在元朝的主要身份是什么？忽必烈汗的侍从。

元朝宫廷吃饭用什么？**刀子和手。** 蒙古贵族不用筷子。

喝茶呢？元代宫廷以马奶酒和发酵奶茶为主，清饮茶是江南汉族文人的习惯。一个在蒙古宫廷生活了17年的外国人，日常喝的是马奶酒而不是龙井——这有什么奇怪的？

**他没记录的东西，恰恰反映了他真实的社交圈：蒙古宫廷的毡帐，不是江南士大夫的书房。**

## 「他没提汉字」—— 他根本就不需要汉字

马可·波罗在元朝使用的语言几乎可以肯定是波斯语。

元朝是一个蒙古人统治、大量使用色目人（中亚和波斯穆斯林）作为行政官员的帝国。波斯语是元朝的国际商业语言和行政辅助语言——《元史》中多次提到"回回文字"（波斯文）用于财政和外交文书。马可·波罗的父亲和叔叔第一次来中国时，就是通过波斯语翻译与忽必烈交流的。

一个用波斯语在元朝工作和生活了17年的外国人，不需要学汉字。他看到的政府公文是蒙古文和波斯文译本。他连汉字长什么样都未必见过几次——不提汉字，恰如其分。

## 「扬州为官三年」—— 这是一个美丽的误会

《行纪》中马可·波罗声称在扬州担任了三年地方官。扬州的地方志里没有他的名字。大多数学者认为他夸大了自己的官职——但这恰好符合他的性格，也符合当时的历史语境。

马可·波罗是一个商人。商人的天性是什么？吹嘘。元朝也确实会委任色目商人临时管理地方盐务和税收——他很可能参与过扬州的商业管理，只是在口述回忆录时把自己"升"了官。这完全是商人式的夸张——把"经手的事"说成"管的城"。

**换个角度想：他要是真的在凭空编造，何必挑扬州这种有地方志可查的城市？** 直接说自己在大都当宰相，没人能对质。正因为他说了扬州——一个可以被证伪的地方——反而说明他真的在那里待过，只是吹了点牛。

当然，诚实地说：马可·波罗有没有来过中国，学术界的争论至今没有完全消停。大英图书馆的 Frances Wood 就写过一本《马可·波罗到过中国吗？》，系统地质疑过。质疑的核心不是某一条证据，而是整本《行纪》中大量"应该提到却未提到"的空白——除了长城、筷子、茶、汉字，还有缠足、印刷术、鸬鹚捕鱼等等。

这些质疑是严肃的，值得尊重。但我个人的判断是：**这些空白，大部分用"蒙古宫廷视角"可以解释；而那些"不应该知道却知道"的细节——比如纸币的材质和流通区域、杭州桥数的量级、云南使用贝币——是坐在热那亚监狱里编不出来的。**

**马可·波罗到底有没有来过中国？** 我个人倾向于：来过。他说的绝大多数内容可以与同期中文和波斯文献互证，他犯的错误都是"当事人会犯的错"而不是"没来过的人编造的错"。

但这个问题也许没那么重要。真正重要的是——他那本在监狱里口述的小书，隔着两个世纪，推动了一个时代。哥伦布带着它发现了新大陆。欧洲的制图师根据它的描述，第一次在地图上标出了"契丹"和"西潘戈"。一个威尼斯商人的见闻，最终重塑了全人类对世界的想象。

**这才是丝路上最神奇的事：信息比货物走得更远。**
